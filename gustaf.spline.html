gustaf.spline.ffd#

Freeform Deformation!

Adaptation of previous implementation in internal python package gustav by Jaewook Lee.

class gustaf.spline.ffd.FFD(mesh: Vertices | Edges | Faces | Volumes | None = None, spline: Bezier | RationalBezier | BSpline | NURBS | None = None)[source]#

Bases: GustafBase

__init__(mesh: Vertices | Edges | Faces | Volumes | None = None, spline: Bezier | RationalBezier | BSpline | NURBS | None = None)[source]#

Free-form deformation is a method used to deform an object by a deformation function. In our case the object is given via a mesh, the currently supported mesh-types are given by the variable gustaf._typing.MESH_TYPES, and the deformations function by a spline, supported splines are given by the variable :py:const.:gustaf._typing.SPLINE_TYPES. The splines parametric dimension will be scaled in to a unit-hypercube as well as the original meshes vertices. The outline of the resulting mesh is given by the physical space of the spline.

The FFD class provides functions to modify the spline by completely overwriting the spline whole spline or parts of it. To obtain the deformed mesh mapped into the latest spline, retrieve the mesh attribute.

Please not that even though an object of the class can be initiated without a mesh, it is not possible to compute the deformation without one. Please ensure that at least a mesh is defined before retrieving the (deformed) mesh. If only a mesh is provided a default spline where the geometric dimensions have the bounds of the mesh is defined.

A previously available partial FFD is currently not implemented, and is planned to be implemented in a separate class (LocalFFD).

Parameters:

  • mesh (Optional[MESH_TYPES]) – Mesh used in the FFD. Defaults to None.

  • spline (Optional[SPLINE_TYPES]) – Spline used in the FFD. Defaults to None.

  • Attributes (Class) –

  • ---------------- –

  • _spline (SPLINE_TYPES) – Internal current spline

  • _mesh (MESH_TYPES) – unscaled base mesh

  • _q_vertices (np.ndarray (n, dim)) – Scaled vertices of the base mesh

Return type:

None
